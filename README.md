# 🛒 React Shopping Cart App

A dynamic e-commerce shopping cart built with React, powered by the FakeStore REST API.

## 🚀 Live Demo
> Deploy on Vercel — see deployment section below

## ✨ Features
- Fetches 20+ real products from FakeStore API
- Global cart state with React Context API + useReducer
- Add/remove items, quantity control
- Cart total & item count badge
- Filter by category
- Fully responsive (CSS Grid + Flexbox)
- Optimised with React.memo and useMemo

## 🛠️ Tech Stack
`React` `Context API` `useReducer` `CSS Modules` `FakeStore REST API` `Vercel`

## ⚙️ Setup & Run

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/react-shop.git
cd react-shop

# 2. Install dependencies
npm install

# 3. Start development server
npm start
```
Open [http://localhost:3000](http://localhost:3000)

## 🚢 Deploy on Vercel
```bash
npm install -g vercel
vercel
```
